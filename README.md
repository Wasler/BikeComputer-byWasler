# BikeComputer-byWasler
Hi everyone, I'm writing this from a translator, so yes, in short, this is my project, not a professional one, so there are flaws and inconsistencies. Basically, I made a bike computer based on the ESP32C3. I just really wanted to find and make a bike computer based on this microcontroller, but I didn't find a single interesting project and decided to make my own using AI. Basically, all you need for the project is 1. a 5x7 cm breadboard, 2. lots and lots of wires, 3. a 9.6-inch OLED screen, 4. an ESP32C3 super mini, 5. one pushbutton, 6. a toggle button for power, 7. a TP4056 charging controller, 8. a digital Hall sensor, 9. a lithium-ion battery or any other battery that fits in the case. ADDITIONALLY: and lastly, it's not required, but you can print it on a 3D printer. printer housing
